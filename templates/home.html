Home Page

Welcome to Eco-Chester, the ulltimate guide for being environmentally friendly in Manchester!

There are several aspects to this website: the maps, the tradeshop, the waste tracker and the chat forum.

Maps

These maps allow the user to see the locations nearby in which they can recycle, while also showing eco-friendly businesses. So next time you plan to go out to eat, take a look at the map for a more eco-friendly option.

Chat Forum

In the chat forum users will be able to ask questions or start discussions about certain topics.

Waste Tracker

The waste tracker will help the user reduce their non-recyclable waste output by showing them statistics about their weekly waste.

Trade Shop

The trade shop enables users to trade used items that they no longer need. This helps to reduce waste as these items are no longer being thrown away.
